十分钟使用hexo搭建博客

今天第一次使用hexo搭建个人博客，被hexo的快熟上手，和简单的操作吸引。让你有更多的时间关注与写作。下面就利用十分钟在本地搭建hexo个人博客。

准备

  1. 首先在本地安装npm包管理器(安装nodejs的时候会默认安装);
  2. 通过npm全局安装hexo，npm install hexo-cli -g

开始搭建

初始化

在命令行中进入你要存放的路径
hexo init blog
执行完成之后会在当前目录下生成blog目录，如图

启动

没错，就是这么快！

你就会看到启动成功的日志，然后访问 http://localhost:4000 就会跳转到初始化的首页(source/_posts目录下)。至此，最简易的一个博客网站已经搭建成功！下面开始码字吧！

创建文档

当然开始之前需要新建一个文档。
hexo new “postname”
或者创建一个新的页面
hexo new page pagename

生成静态页面

创建完文档之后，需要将文档编译成html。
hexo generate

生效

hexo server
重新打开刚才的url，就可以看到我们刚才创建的文档了。

至此，你应该可以专心去创作了！如果觉得默认主题不好看，还可以修改主题哦，只需三步：

  1. 找到心仪的主题
  2. clone到themes目录下
  3. 修改配置文件_config.yml最下面的配置theme: theme name
